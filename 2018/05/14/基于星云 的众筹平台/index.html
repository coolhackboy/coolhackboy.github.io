    基于星云开发的众筹平台

    星云链是致力于构建可持续升级良性生态的下一代公链

    官方地址： https://nebulas.io/cn/index.html

    准备工作

    web-wallet

    官方钱包，可以使用钱包进行测试网，主网合约部署

    地址：https://github.com/nebulasio/web-wallet

    git clone https://github.com/nebulasio/web-wallet.git

    钱包插件-WebExtensionWallet

    chrome浏览器钱包插件，我们也下载下来，pc端进行交易要用到

    git clone https://github.com/ChengOrangeJu/WebExtensionWallet.git

    刚进来的同学可以学习一下怎么编写合约，星云生态很完善，文档资料也比较全，所以学起来真的很轻松

    相关学习文档

    学习文档：https://github.com/nebulasio/wiki/tree/master/tutorials

    开发知识库：http://nasfans.io/forum.php?mod=viewthread&tid=117

    官方demo：https://github.com/15010159959/super-dictionary 作者也是学这个demo过来的

    前端页面

    核心代码join.html

    nebPay.call方法进行调用合约接口，我这里调用合约的接口为addStarPlan，合约交易完毕后监听返回函数cbPush，对数据进行查询

    注意：使用nebPay需要安装 钱包插件WebExtensionWallet
